GiMoveis

Site institucional responsivo desenvolvido para apresentar a GiMoveis, empresa de moveis planejados em Ubajara, Ceara.

O projeto traz uma pagina principal com video de apresentacao, informacoes sobre a empresa, diferenciais, galeria de trabalhos realizados e area de contato com acesso direto ao WhatsApp, Instagram e localizacao.

A galeria possui imagens organizadas por sequencia, permitindo navegar entre fotos relacionadas de um mesmo projeto. As imagens tambem podem ser ampliadas para visualizacao em tela maior.

Tecnologias utilizadas

HTML
CSS
JavaScript

Estrutura do projeto

index.html
Arquivo principal da pagina.

style.css
Estilos visuais, responsividade, galeria e modal de imagem ampliada.

script.js
Interacoes da pagina, menu mobile, animacoes, carrossel da galeria e visualizacao ampliada.

assets
Pasta com imagens, logo e video utilizados no site.
